# 生鲜同城

禹邻优鲜单门店生鲜同城配送系统，包含微信小程序客户端、Spring Boot 后端服务和 Vue 管理后台。

## 项目结构

- `client-wechat`：微信小程序客户端。
- `server`：Spring Boot 后端接口服务。
- `admin-web`：Vue 3 管理后台。
- `docs`：PRD、页面流程、接口清单和生产检查文档。
- `design`：Stitch 原型包与截图资源。

## 当前能力

- 微信小程序：商品浏览、分类、搜索、购物车、地址地图选点、预约配送、微信登录、微信支付、订单、退款申请、个人中心。
- 后端：用户登录与资料同步、商品/分类/库存、地址、购物车、订单、支付、退款、配送时段、门店配置、文件上传。
- 管理后台：商品管理、图片上传、订单处理、售后退款审核、配送时段、门店配置。

## 本地启动

1. 后端：进入 `server`，执行 `./mvnw.cmd spring-boot:run`。
2. 管理后台：进入 `admin-web`，执行 `npm install` 后执行 `npm run dev`。
3. 小程序：使用微信开发者工具打开 `client-wechat`。

## 生产发布

发布前必须完成 `docs/06-production-checklist.md` 中的环境变量、微信支付、小程序合法域名、后台账号、数据目录和构建检查。
